# reverse

Reverses array.

:docs-badges{ :badges='[]' }


## Data First

```js [light]
P.reverse(arr);
```

:docs-parameters{ :params='[{"name":"array","description":"the array"}]' :returns='{"name":"Object"}' }

```js
P.reverse([1, 2, 3]); // [3, 2, 1]
```

## Data Last

```js [light]
P.reverse()(array);
```

:docs-parameters{ :params='[]' :returns='{"name":"Object"}' }

```js
P.reverse()([1, 2, 3]); // [3, 2, 1]
```
