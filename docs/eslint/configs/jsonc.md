## Custom Overrides

If you want to override or add more rules you can pass it to the `overrides` property.
For example:

```js [eslint.config.js]
import { vinicuncaESLint } from '@vinicunca/eslint-config';

export default vinicuncaESLint(
  {
    jsonc: {
      overrides: {
        'jsonc/array-bracket-spacing': 'off',
      }
    },
  }
);
```
